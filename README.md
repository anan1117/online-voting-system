# 玉山銀行 - 線上投票系統 (實作題)

本專案採用前後端分離架構開發。

技術棧
- **後端**: Java Spring Boot 3, JDBC Template, H2 Database
- **前端**: Vue 3 (Composition API), Vue Router
- **資料庫**: H2 In-Memory Mode (含 Transaction 事務處理)

專案結構
- `/DB`: 存放 DDL 與 DML 腳本 (符合題目需求)
- `/vote-backend`: Spring Boot 後端代碼
- `/vote-frontend`: Vue.js 前端代碼

如何執行
1. **後端**: 以 Eclipse 匯入 Maven 專案，執行 `OnlineVotingSystemApplication.java`。
2. **前端**: 進入 `vote-frontend` 目錄，執行 `npm install` 與 `npm run dev`。
